Here is the demonstration of command line arguments
This is a little notes app where you can add notes with title and body.
There are 4 commands add, remove, list and read.

1. add command: this command takes two arguments title and body.
2. remove command: this arg will take title as an arguments
3. list command : this command will list the title of all the notes present in the directory.
4. read command : this command will read the specific note you want. this command takes title as an arguments
